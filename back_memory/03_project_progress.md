| TypeScript 构建错误        | 已修复过 | store 输入类型、表单类型、Settings 返回类型等         |

| `institution:update` 不支持安全局部更新 | 已修复 | 主进程按 patch 语义构造 update data，只更新明确传入字段 |
| `advisor:update` 不支持安全局部更新 | 已修复 | 切换联系状态时只更新 `contactStatus`，不会清空导师姓名、邮箱、研究方向 |
| 详情页 assets/interviews 数据加载不完整 | 已修复 | `institution:getAll` 改为深层 include `advisors: { include: { assets: true, interviews: true } }` |
| 新增 Asset/Interview 后详情页不刷新 | 已修复 | `addAsset`/`addInterview` 成功后调用 `loadInstitutions()` 刷新全局数据 |
| 邮件预览变量填充值 XSS | 已修复 | `renderPreviewText` 中变量名和填充值均通过 `htmlEscape()` 转义 |
| LaTeX 编译 shell 注入和路径解析 | 已修复 | 改用 `spawn` + `path.dirname()` 替代 `exec` + 字符串拼接 |
| 日期格式化未统一，无效日期可能白屏 | 已修复 | 新增 `parseValidDate`/`formatDateSafe` 统一工具 |
| 导入导出不是完整备份 | 已修复 | 新增 `backup:exportAll`/`backup:importAll` IPC |
| 任务 dueDate 可空与 schema 冲突 | 已修复 | `task:update` 中 dueDate 为 null/空时跳过该字段 |
| 外部链接打开无协议限制 | 已修复 | `setWindowOpenHandler` 仅允许 http/https/mailto 协议 |
